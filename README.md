# Angular Hello World

This template provides a minimal Angular standalone setup.

It is the starting point for the Graftcode quick-start guide:
[Connect an Angular frontend to a backend](https://github.com/grft-dev/graftcode-quick-start-guide/blob/main/1-connect-frontend-to-backend/angular.md).

## Prerequisites

- [Node.js](https://nodejs.org/) installed locally
- [Angular CLI](https://angular.dev/tools/cli) installed (`npm install -g @angular/cli`)

## Getting started

```bash
npm install
ng serve
```

Open the URL shown in the terminal (typically [http://localhost:4200](http://localhost:4200)).
